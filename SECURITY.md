# Security Policy

## Supported Versions

The latest `main` branch and the latest GitHub release are supported for security fixes.

## Reporting a Vulnerability

Please do not open public issues for sensitive security problems.

Report vulnerabilities by opening a private security advisory on GitHub, or contact the maintainer through the GitHub profile associated with this repository.

Please include:

- Affected version or commit.
- Steps to reproduce.
- Expected and actual behavior.
- Impact and any suggested mitigation.

## Notes

This project is currently a local-first static web app. The development server includes local-only endpoints for editing study data and uploading images. Do not expose the Vite development server directly to the public internet.
