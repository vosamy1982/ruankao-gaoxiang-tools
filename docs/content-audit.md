## 自动检查范围

审计脚本会阻止以下内容进入通过状态：

- 数据根节点、必填字段、字段类型或关联 ID 无效。
- 章节、考点、知识域、过程组或过程 ID 重复。
- 同一章节中的考点标题重复，或考点正文完全重复。
- 正文超过 1,200 字符、单行超过 600 字符或长 Markdown 引用超过 300 字符。
- 正文包含占位内容、疑似转载来源标记或外部链接。
- 图片使用远程地址、越出 `/images/` 路径或缺少对应的已提交文件。

明确授权的内容如果触发审阅规则，可由维护者在 `scripts/data-audit-allowlist.json` 中登记精确的规则代码、数据位置和批准原因。结构错误、缺失文件和占位内容不应加入例外；已不再对应实际发现的过期例外也会导致审计失败。

## 边界

自动规则只能识别结构错误和明显风险，无法证明内容一定原创，也无法判断改写是否仍然过度接近原始资料。涉及新增学习内容或图片时，维护者仍需人工确认表达来源、授权范围和再分发条件。
