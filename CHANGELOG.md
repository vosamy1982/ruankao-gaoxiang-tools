# Changelog

本项目遵循语义化版本的基本习惯：功能增强使用小版本，缺陷修复和维护更新使用补丁版本。

## v0.10.0 - 2026-06-22

- 在复习中心增加今日学习计划，可选择每日 5、10、15 或 20 个考点。
- 今日队列按智能复习优先级生成，并在日期变化时自动开始新计划。
- 支持逐项完成、进度显示和刷新后继续，完成记录与考点掌握状态相互独立。
- 增加今日计划持久化、日期切换、目标调整和恢复逻辑测试。
- 将 GitHub Actions 的 `actions/checkout` 更新到 v7。

## v0.9.2 - 2026-06-19

- 更新 `lucide-react` 和 `typescript-eslint` 小版本依赖。
- 增强桌面端下拉导航的键盘可访问性，支持 `Enter`、`Space`、方向键、`Home`、`End` 和 `Escape`。
- 为导航按钮和菜单项补充可见焦点样式。
- 新增项目变更日志。
- 更新前端性能基线。

## v0.9.1 - 2026-06-16

- 修复下拉菜单被矩阵页 sticky 表头遮挡的问题。
- 提升导航层级，同时保持详情弹窗位于导航之上。

## v0.9.0 - 2026-06-16

- 将顶部导航改为桌面端下拉菜单。
- 移动端改为分组纵向菜单，避免导航按钮换行挤压。
- 增加当前位置提示。
- 刷新 README 截图和性能基线。

## v0.8.0 - 2026-06-16

- 将顶部导航按 ITTO 查询、学习复习、资料速查和内容维护进行分组。
- 降低所有视图按钮同时显示造成的首屏拥挤。

## v0.7.0 - 2026-06-16

- 将快速复习能力独立为复习中心。
- 复习队列支持智能复习、学习中、未学习和收藏模式。
- 保持学习记录与考点内容数据分离。

## v0.6.0 - 2026-06-15

- 增加快速复习模式。
- 支持基于学习记录生成复习队列。

## v0.5.0 - 2026-06-13

- 增加本地学习记录。
- 支持考点收藏、学习状态、进度统计、记录导入导出和清空。
